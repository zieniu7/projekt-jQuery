# projekt-jQuery

## Opis
Aplikacja do zarządzania notatkami z użyciem jQuery.

## Funkcje
- Dodawanie, edycja i usuwanie notatek
- Animacje (fadeIn, slideUp)
- Walidacja formularza (jQuery Validate)
- Tabela DataTables (sortowanie, wyszukiwanie, paginacja)
- Tryb ciemny
- Datepicker (jQuery UI)

## Technologie
- jQuery
- DataTables
- jQuery Validate
- Bootstrap

## Uruchomienie
Otwórz plik `index.html` w przeglądarce
